## 💡 Project Overview

This coursework covers the evolution of vision tasks, starting from pixel-level probabilistic segmentation and moving into real-time deep learning-based object detection.

## 🚀 Assignments Summary

| Assignment | Task | Methodology / Technique |
| --- | --- | --- |
| **Assignment 1** | Image Segmentation | **Naive Bayes Classifier**: Probabilistic pixel-level classification to isolate objects from the background. |
| **Assignment 2** | Object Detection | **YOLO (You Only Look Once)**: Implementation and performance comparison of two distinct YOLO versions. |

---

## 🛠 Technical Stack

* **Core Vision:** OpenCV, SciPy, Matplotlib
* **Statistical Modeling:** NumPy, Scikit-learn (for Naive Bayes)
* **Deep Learning:** PyTorch / TensorFlow (for YOLO implementations)
